13 ## To run Encryption Platform
14 ************************************
15 1. Open a Linux Terminal.
16 2. Navigate to the Encyption Platform Directory.
17 3. Run the "make" command.
18 4. An executable called "encyptionPlatform" will be created.
19 5. Use "./encyptionPlatform" to run the encryption platform. (if no input parameters are specified, a help menu will be displayed)
20 6. A list of input parameter and default values:
21 
22 | Parameter | Description | Default Value |
23 |-------------------- |------------------------------------------------- |------------------ |
24 | -h | Print out the help menu | |
25 | -f or --filename | Path to an ASCII file to encrypt/decrypt | None |
26 | -F | Path to a hex file to encrypt/decrypt | None |
27 | -k or --key | The encryption key | 128 bit NULL key |
28 | -K | The encryption key (in hex) | None |
29 | -i or --IV | The Initialisation vector | 128 bit NULL IV |
30 | -I | The Initialisation vector (in hex) | None |
31 | -p or --plaintext | Plaintext to encrypt | None |
32 | -P | Plaintext to encrypt (in hex) | None |
33 | -c or --ciphertext | Ciphertext to decrypt | None |
34 | -C | Ciphertext to decrypt (in hex) | None |
35 | -m or --mode | Mode of encryption/decryption (ecb, cbc or cfb) | ecb |
36 | -v or --verbose | Print verbose output | Flag not set |
37 | -e or --encrypt | Encrypt plaintext or file | Flag set |
38 | -d or --decrypt | Decrypt ciphertext or file | Flag not set |
39 | -T | Set if input text is a hex string | Flag not set |
